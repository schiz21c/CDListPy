# CDListPy
Catalog your CD Collections

## Requirements
- Python 2.x.x
- wxPython
- pywin32
- yaml

## Deprecated
- pywin32 related functions

## Installation
- wxPython
```
pip install wxpython
```

- Run CDListPy
```
./CDListPy.py
```

- Optional: Run setup_img2py if you want to change the icon. This will make "res.py" file
```
./setup_img2py.py
```
