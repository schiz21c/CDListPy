# CDListPy
CDList for Python

## Requirements
- wxPython
- pywin32
- yaml
- py2exe (optional)

## Known Issues
- Works only in Windows

## Usage
1. Run img2py script.
``` shell
python setup_img2py.py
```
2. (optional) Run py2exe script to create EXE file.
``` shell
python setup_py2exe.py
```
3. Run CDListPy
``` shell
python CDListPy.py
```
