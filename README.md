# CDListPy
CDList for Python

## Requirements
- wxPython
- pywin32
- yaml
- py2exe (optional)

## Known Issues
- Works only in Windows

## Usage
- Run img2py script at first. This will make res.py (Only once)
```
python setup_img2py.py
```

- Run main script
```
python CDListPy.py
```

- (optional) Run py2exe script to create EXE file
```
python setup_py2exe.py
```
