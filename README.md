# CDListPy
CDList for Python

## Requirements
- wxPython
- pywin32
- yaml
- py2exe (optional)

## Known Issues
- Works only in Windows

## Usage
- Run img2py script before CDListPy. (Only once)
``` shell
python setup_img2py.py
python CDListPy.py

```
- (optional) Run py2exe script to create EXE file
``` shell
python setup_py2exe.py
```
